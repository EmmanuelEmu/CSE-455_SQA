.. container:: teacher-details Detailed information about a teacher. This template extends the ``main.html`` template and is used to display detailed information about a teacher. It includes the teacher's ID, name, rank, department, description, email, and contact number. Additionally, it provides options to delete or update the teacher's information, although these options are currently commented out. Example Usage: .. code-block:: html {% extends 'main.html' %} {%block content%}
Teacher ID : {{teacher.id}}

{{teacher.name}}

{{teacher.rank}}
{{teacher.dept}}

{{teacher.description}}

Contact Information
Email : {{teacher.email}}
Contact Number : {{teacher.phone}}

{% comment %} Delete {% endcomment %}
{% comment %} Update {% endcomment %}
{%endblock%}
